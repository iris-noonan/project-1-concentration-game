Concentration matching game

Intructions: Start the game by pressing play. The timer will start. Select a square to turn and show a Studio Ghibli film. Select another square to try to find the matching film. If it's a match you win, if not the square will return to the original form until you find two matching films. Continue to play until you match all the films and all the squares are revealed. Try to complete this in the least amount of time and moves!
